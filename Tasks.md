####Create variables
In `src/scss/constants.scss` create variables for given colors and font sizes. 
Variable looks like this: `$primary: #30004A;`  
Colors
- `#30004A`
- `#DD4913`
- `#737373`
- `#E5E5E5`
- `#fff`
- `#000`  

Font sizes
- small `0.8125rem;`
- medium `1.125rem;`
- large `1.5rem;`
